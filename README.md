## 📖 简介 (Introduction)

**Poop Tracker** 并不是一个普通的记录器，它的核心理念是：**关注肠道健康也可以是一件优雅且有趣的事情。**

无需下载 App，无需注册账号，打开网页即可使用。所有数据仅存储在你的设备本地，绝对私密。

## ✨ 功能亮点 (Features)

* **⏱️ 沉浸式计时器**
    * 点击开始，中央的 💩 会随着呼吸动画律动，陪伴你的“释放”时光。
    * 大字体显示时长，操作简单直观。

* **📊 多维健康评分系统 (Health Score Algorithm)**
    * 基于 **布里斯托大便分类法 (Bristol Stool Scale)**、排便顺畅度 (Effort) 以及 颜色 (Color) 进行综合计算。
    * 自动生成 **0-100分** 的健康评分，让你一眼掌握健康状况。

* **🎨 优雅的 UI 设计**
    * 全套紫色治愈系配色，圆角卡片设计。
    * 动态 Emoji 反馈：根据你的选择，实时显示对应的粑粑形态（从 🌑 石头到 🐍 完美香肠）。

* **🔒 隐私第一 (Privacy First)**
    * 采用 `Local Storage` 技术。
    * **数据只保存在你的浏览器中**，不会上传到任何服务器。

* **📱 移动端适配**
    * 专为手机浏览器优化，体验如同原生 App。

## 🚀 如何使用 (How to Use)

1.  打开网页。
2.  点击 **"START POOPING"** 开始计时。
3.  结束后点击 **"FINISH"**。
4.  滑动滑块评价顺畅度，选择形态和颜色。
5.  点击 **"SAVE 💩"**，系统将自动计算分数并保存到历史记录。

## 🛠️ 技术栈 (Tech Stack)

* **HTML5**: 语义化结构
* **CSS3**: Flexbox 布局, CSS Variables, Keyframe 动画
* **JavaScript (Vanilla)**: 原生逻辑，无第三方库依赖
